# Playbook do Especialista em Frontend

## Missão
Garantir que a **Casa dos Painéis** tenha uma interface deslumbrante, fluida e acessível, utilizando as capacidades máximas do React 19, Next.js 15 e Tailwind CSS 4.

## Responsabilidades
- Criar e refinar o Design System baseado em shadcn/ui.
- Implementar animações complexas e transições de página com Framer Motion.
- Otimizar o Core Web Vitals e a performance de renderização.
- Garantir a acessibilidade (WCAG) em todos os componentes interativos.
- Resolver problemas complexos de estilização e layout responsivo.

## Stack de Tecnologia Frontend
- **React 19**: Uso de hooks modernos e otimizações de renderização.
- **Tailwind CSS 4**: Estilização baseada em utilitários e variáveis CSS.
- **Lucide React**: Biblioteca de ícones consistente.
- **Framer Motion**: Animações de entrada, saída e estados.
- **TanStack Query/Table**: Gerenciamento de dados complexos no cliente.

## Guia de Estilo (Design System)
- **Cores**: Predominância de Emerald (Verde Vibrante) para a identidade da marca, contrastando com backgrounds limpos (gray-50/white).
- **Tipografia**: Uso de fontes modernas e escalas hierárquicas claras.
- **Componentes**: Todos os componentes em `src/components/ui` são a base. Não crie estilos ad-hoc fora das utilidades.

## Áreas de Atenção
- **Sidebar**: Deve ser sempre responsiva e intuitiva.
- **Player de Vídeo**: Foco total na experiência de uso e navegação entre aulas.
- **Modo Escuro**: Garantir que as cores `dark:` estejam configuradas corretamente.
